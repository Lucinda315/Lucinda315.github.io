Using BioT2Ex

Inroduction

BioT2Ex provides a package or toolkit to conveniently convert the tabular txt file into a well-anotated Excel file. We believe that BioT2Ex could facilitate the downstream data interpretation of high-throughput genomic sequencing analysis.

BioT2Ex now supports the results of the RNA-seq analysis,Chip-seq analysis and Wes data analysis.

Running BioT2Ex in R

Once installed the package, just go to your analysis directory and use the functions.

Render the output

Select the appropriate render function according to the output results, and conduct a preliminary analysis and processing of your output results and format conversion. Currently BioT2Ex’s render function supports output from edgeR, DESeq2 and ChIPseeker.

renderedgeR

The output results were sorted in descending order according to the size of foldchange, annotations representing up-regulation or down-regulation of expression were added, geneId was added, then GO enrichment analysis was further conducted, GO number was output, and Genebank link of each gene was added.A list of table header comments is also generated in this step.

The original expression difference analysis data are from the paper.

Output effect:

renderDESeq2

The output results were sorted in descending order according to the size of foldchange, annotations representing up-regulation or down-regulation of expression were added, geneId was added, then GO enrichment analysis was further conducted, GO number was output, and Genebank link of each gene was added. A list of table header comments is also generated in this step.

The original expression difference analysis data are from the paper.

Output effect:

renderChIPseeker

Add “ENSEMBL”,”SYMBOL”,”GENENAME” and a list of table header comments is also generated in this step.

The original data is downloaded from the GEO database.(GSE144195_combined_peaks_with_CTCF_motif.bed.gz)

Output effect:

Add gene description

A function to add description of genes by geneId.

addDescription

To use this function you need to enter the taxID of the species and the column number of geneId.

Output effect:

Write the data into excel

This function inputs the previously processed data into workbook and name the workbook.

writeOut

Output effect:

Add header comments

Add header comments to the output result.

addHeadAnnotation

This function adds comment information to the header, either using default arguments or custom comment content.

Output effect:

Add feature to the excel

This function modifies the tabular style of the output.

addFeature

This function will set a uniform font size for the output results, and add a gradient representing the expression level and size for the foldchange column. Red represents up and blue represents down.

Output effect:

Other functions

These functions are included in some main function functions to further improve their effects.

correctNumber

This function can select the numeric column in the file and makes it displays correctly.

showInExcel

A function to open selected directory or file in excel, this base fuction is copied from o function of yulab.utils(package).

renderPackage

This is a function integration function, which can directly complete the above analysis steps. Currently, only edger, deseq2 and chipseeker output results are supported.
